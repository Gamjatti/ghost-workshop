# 유령제작소 (GHOST WORKSHOP) — 디자인 가이드

> AI 디자인의 마지막 20%를 다루는 HCI 기반 도구. 다크 네이비 베이스 + 로고에서 추출한 테라코타·스틸블루 듀얼 액센트, 절제된 그라데이션과 의미 있는 모션.

### 1-3. 시그니처 그라데이션
```
--grad: linear-gradient(120deg, #5B8FD0 0%, #DA6A4A 100%);  /* 스틸블루 → 테라코타 */
```
강조 단어, 브랜드 마크, 80/20 바, 카드 외곽 글로우에 사용. **로고의 두 포인트 색을 잇는 그라데이션 = 브랜드 시그니처.**

### 1-4. 카테고리 기능색 (라벨 전용)
| 카테고리 | 색 |
| --- | --- |
| 마지막 20% | `#DA6A4A` 테라코타 |
| Fix Lab | `#5B8FD0` 스틸블루 |
| Worknote | `#8FB7E0` 라이트블루 |

> 규칙: **메인 액센트는 테라코타 하나로 고정.** 스틸블루는 그라데이션·보조·카테고리에만. 순수 검정(`#000`)·순수 흰색 금지(깊이감 손실).

## 2. 타이포그래피

| 역할 | 폰트 | 비고 |
| --- | --- | --- |
| 국문 본문·제목 | **Pretendard** | 기본 |
| 영문·숫자 디스플레이 | **Space Grotesk** | `20%`, `Blog`, `BETA` 등 라틴/숫자에 자동 적용 |
| 보조(영문 라벨) | Inter | 폴백 |

```
--font:    "Pretendard", system-ui, sans-serif;       /* 본문 */
--display: "Space Grotesk", "Pretendard", sans-serif; /* 큰 제목·숫자 */
--mono:    "Space Grotesk", "Inter", ...;             /* 라벨·메타 */
```
- 제목: `letter-spacing: -0.03 ~ -0.045em` (타이트하게)
- 본문: line-height 1.65, 최대폭 `64ch`
- 강조 단어는 `.hl` → 시그니처 그라데이션 글자 (천천히 흐르는 애니메이션)

- 파일: `icon.svg`(아이콘), `logo.svg`(다크용 풀 워드마크)

## 5. 컴포넌트

- **버튼**
  - Primary `.btn--accent`: 솔리드 테라코타 + 어두운 글자(`#241008`), 글로우 + 상단 하이라이트
  - Ghost `.btn--ghost`: 투명 + 1px 보더, 호버 시 테라코타
  - 라벨은 1~2단어, 데스크탑에서 줄바꿈 금지
- **카드** `.pcard` / `.cat`: 글래스(backdrop-blur) + 그라데이션 외곽선, 호버 시 부양 + 테라코타 링 + 커서 스포트라이트
- **태그** `.tag--20/fix/note`: 카테고리 색 outline pill
- **eyebrow**: 글래스 pill 라벨 (남용 금지, 섹션 3개당 1개 이하)
- **80/20 바** `.ratio`: 빗금 80% + 그라데이션 20% — "마지막 20%" 시그니처 장치
- **CTA 밴드** `.ctaband`: 깊은 그림자 + 내부 글로우
- **나브**: 글래스 + 스크롤 시 진해짐, Blog 호버 시 카테고리 드롭다운(쭈루룩)

## 6. 모션 원칙

> 모든 모션은 `transform`·`opacity`만, `prefers-reduced-motion`에서 자동 정지. 데스크탑(마우스/트랙패드)에서만 포인터 효과 작동.

| 효과 | 설명 |
| --- | --- |
| 스크롤 리빌 | 요소가 아래에서 부드럽게 등장 (stagger) |
| 커서 글로우 | 마우스를 관성 있게 따라오는 빛 |
| 카드 스포트라이트 | 카드 위 커서 위치에 오렌지 하이라이트 |
| 자석 버튼 | 주요 버튼이 커서 쪽으로 살짝 끌림 |
| 무빙 메시 | 배경 오로라가 천천히 흐름 |
| 그라데이션 글자 | 강조어 그라데이션이 흐름 |
| 히어로 3D | "AI 레이아웃 80% + 선택된 20% 완성" 컨셉 3D (Three.js) |

원칙: **모션은 반드시 의미가 있어야 함**(위계·피드백·스토리). 장식용 무한 루프 남발 금지.
