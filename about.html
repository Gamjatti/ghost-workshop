GHOST WORKSHOP / 유령제작소

인간의 의도와 AI의 생성이
만나는 지점을 만듭니다.

유령제작소는 인간의 의도와 AI의 생성 능력이 만나는 지점을 연구하고, 이를 실제 작업 도구로 구현하는 팀입니다. 우리는 AI가 모든 것을 대신하는 미래보다, 인간의 의도와 판단이 더 잘 전달되는 작업 환경을 만듭니다.

AI는 만들 수 있습니다.
하지만 원하는 것을 정확히 전달하는 일은
여전히 어렵습니다.

생성형 AI는 빠르게 결과물을 만들어냅니다. 그러나 실제 업무에서는 생성 이후의 수정, 판단, 검증, 의도 전달이 더 큰 병목이 됩니다. 유령제작소는 이 병목을 해결하기 위한 도구와 시스템을 만듭니다.

우리는 생성 이후의
작업 경험을 설계합니다.

유령제작소는 AI 디자인 후처리 도구와 도메인 특화 AI Agent 시스템을 개발합니다. 핵심은 AI가 만든 결과물을 사람이 다시 붙잡고, 판단하고, 조정하여 실제 업무에 쓸 수 있는 수준으로 끌어올리는 과정입니다.

PRODUCT 01

AI 디자인 후처리 도구

AI가 만든 결과물을 실제 업무에 쓸 수 있는 수준까지 다듬는 작업 흐름을 연구합니다. 전체 재생성이 아니라 원하는 부분을 중심으로 수정합니다.

PRODUCT 02

도메인 특화 AI Agent

특정 산업의 반복 문서, 검색, 요약, 검증 과정을 자동화하는 AI 시스템을 구축합니다. 도메인의 맥락을 이해하는 에이전트를 지향합니다.

AI-Native Workflow를
실제 도구로 구현합니다.

기존 도구에 AI 기능을 붙이는 것과, 처음부터 AI와 인간의 협업을 전제로 설계하는 것은 다릅니다. 유령제작소는 생성, 수정, 검증, 피드백이 하나의 루프로 이어지는 작업 환경을 지향합니다.

만들 수 있는 팀

유령제작소는 AI·보안 R&D 배경의 풀스택 개발 역량을 기반으로, 자체 SaaS 제품과 도메인 특화 AI 시스템을 함께 구축합니다.

  • 01정보보안 전공 기반의 시스템 이해
  • 02풀스택 개발 역량으로 제품을 직접 구현
  • 03AI Agent 개발 경험과 자체 제품 개발 경험
  • 04HW · OS · Application으로 연결된 지식
  • 05도메인 특화 시스템 구축 경험
